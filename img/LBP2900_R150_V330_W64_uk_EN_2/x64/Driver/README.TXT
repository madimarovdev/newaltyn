1. Scaling

  Some applications may not print correctly if scaling is 
  specified. In this case, please set its scaling at 100%.
  If the application has its own scaling function, please set 
  scaling from the application.

================================================================

2. Page Layout

  Some applications provide a function to print multiple sets of
  copies. However, do not use the function when a setting other 
  than [1 Page per Sheet] is selected for [Page Layout] in the 
  [Page Setup] sheet.
  If the function is used in the application, the data may not 
  be laid out properly.

================================================================

3. Separator Page

  [Separator page] on Windows cannot be printed.

================================================================

4. User Permission on the Installation 
   (Windows 2000/XP/Vista/7/Server 2003/Server 2008)

  Log on as a user with administrator rights to install the 
  printer driver. The Full Control Permission is required for 
  the installation.

================================================================

5. Using Microsoft Office after the Installation

  If you are using the installer, and you do not restart the 
  computer when a message appears prompting you to do so, 
  Microsoft Office may not operate correctly after that. If this
  happens, restart your computer.

================================================================

6. Uninstalling the network printer which is connected to 
   another user 
   (Windows 2000/XP/Vista/7/Server 2003/Server 2008)

  When you execute the [Canon LBP2900 Uninstaller] on the 
  computer that other user have already connected to the Canon 
  LBP2900 network printer servers, information of the Canon 
  LBP2900 network printer that other user connected will not be 
  removed. In this case, other user should remove his Canon 
  LBP2900 printer icon before you execute uninstallation.

================================================================

7. Restriction on the Network Printer 
   (Windows 2000/XP/Vista/7/Server 2003/Server 2008)

  When a network printer is installed on Windows 
  2000/XP/Vista/7/Server 2003/Server 2008, even though you 
  change its connection destination to a local port, the printer
  cannot be used as a local printer.
  If you use the printer as a local printer, uninstall the 
  installed network printer, select a local port, and then 
  install the printer driver anew.

================================================================

8. Restarting Windows with a remaining print job 
   (Windows 2000/XP/Vista/7/Server 2003/Server 2008)

  If you need to restart Windows with a remaining Canon LBP2900 
  print job, pause the print job from the printer folder.
  Then restart Windows.

================================================================

9. Downloading and Installing a Network Printer 
   (Windows 2000/XP/Vista/7/Server 2003/Server 2008)

  When downloading and installing a network printer on Windows 
  2000/XP/Vista/7/Server 2003/Server 2008, you need to install 
  it as a user with administrator rights.
  If the network printer is installed by a user without 
  administrator rights, some functions may be unavailable due to
  the restriction of operating system security.

  [Unavailable functions]
  - The Sound feature of Printer Status Window is not functioned
    properly.
  - Uninstaller cannot be registered in [Add/Remove Programs].
    ([Add or Remove Programs] for Windows XP/Server 2003; 
    [Uninstall a program] for Windows Vista/7/Server 2008 R2; 
    [Programs and Features] for Windows Server 2008)

================================================================

10. Installation in a Multi-user Environment 
    (Windows XP/Vista/7/Server 2008)

  When Canon LBP2900 is installed in Windows 
  XP/Vista/7/Server 2008, do not install it for the user 
  switched by the "Fast User Switching" function of Windows 
  XP/Vista/7/Server 2008.
  When using the "Fast User Switching" function, be sure to log 
  on again before installation.

================================================================

11. The Protected Mode of Internet Explorer 
    (Windows Vista/7/Server 2008)

  If you are using Windows Vista/7/Server 2008 and run Internet 
  Explorer by clicking the icon for [Explanation of Software] 
  etc. in CD-ROM Setup, [Protected Mode] is set to [Off] 
  regardless of the settings.
  If you want to set [Protected Mode] to [On], exit Internet 
  Explorer once, and then run it again manually.

================================================================

12. Displaying the Printer Status Window in a Multi-user 
    Environment (Windows XP/Vista/7/Server 2008)

  If the user is changed with the "Fast User Switching" function
  of Windows XP/Vista/7/Server 2008, the printer status cannot 
  be displayed in the Printer Status Window. To display the 
  status, log on again, then open the Printer Status Window.

================================================================

13. CAPT Windows Firewall Utility

  If you are using an operating system equipped with Windows 
  Firewall, such as Windows XP Service Pack 2, as a print 
  server, communication with the client computers is controlled 
  by Windows Firewall. The [CAPT Windows Firewall Utility] is a 
  utility for configuring Windows Firewall to unblock or block 
  this communication with the client computers.
  The [CAPT Windows Firewall Utility] is located in the 
  [English] - [WF_UTIL] folder in the supplied [LBP2900 User 
  Software] CD-ROM.
  Refer to Chapter 7: "Windows Firewall" in the User's Guide for
  more details on the [CAPT Windows Firewall Utility].

================================================================

14. About the Client-Side Settings for Using the Printer Sharing
    Function

  If you are using Windows XP Service Pack 2 or another 
  operating system equipped with Windows Firewall on the client 
  side, configure the following settings. If you do not setup 
  the following settings on the client side, the printer status 
  may not display properly in the Printer Status Window and you 
  may not be able to access some of the printer functions.

  (If the [User Account Control] dialog box appears while you 
  are specifying the following settings, click [Continue] or 
  [Yes].)

  - For Windows XP
    (1) From the [Start] menu, select [Control Panel], and then 
        click [Network and Internet Connections] -> [Windows 
        Firewall].
    (2) In the [Exceptions] sheet in the [Windows Firewall] 
        dialog box, select the [File and Printer Sharing] check 
        box, and then click [OK].

  - For Windows Vista
    (1) From the [Start] menu, select [Control Panel], and then 
        click [Allow a program through Windows Firewall].
    (2) In the [Exceptions] sheet in the [Windows Firewall 
        Settings] dialog box, select the [File and Printer 
        Sharing] check box, and then click [OK].

  - For Windows 7
    (1) From the [Start] menu, select [Control Panel], and then 
        click [System and Security] -> [Allow a program through 
        Windows Firewall].
    (2) Click [Change settings], select the [File and Printer 
        Sharing] check box, and then click [OK].

  - For Windows Server 2003
    (1) From the [Start] menu, select [Control Panel] -> 
        [Windows Firewall].
    (2) In the [Exceptions] sheet in the [Windows Firewall] 
        dialog box, select the [File and Printer Sharing] check 
        box, and then click [OK].

  - For Windows Server 2008
    (1) From the [Start] menu, select [Control Panel], and then 
        double-click [Windows Firewall].
    (2) Click [Change settings] in the [Windows Firewall] dialog
        box.
    (3) In the [Exceptions] sheet in the [Windows Firewall 
        Settings] dialog box, select the [File and Printer 
        Sharing] check box, and then click [OK].

  - For Windows Server 2008 R2
    (1) From the [Start] menu, select [Control Panel], and then 
        click [Check firewall status] -> [Allow a program or 
        feature through Windows Firewall].
    (2) Click [Change settings] in the [Allowed Programs] dialog
        box.
    (3) Select the [File and Printer Sharing] check box, and 
        then click [OK].

15. Characters for the Printer Name 
    (Windows Vista/7/Server 2008)

  When an "@" character is used in the name of the printer, the 
  printer will become unrecognizable by the operating system 
  because of an invalid printer name thus disabling printing, 
  operation and setup.

================================================================

16. Registering Custom Paper Sizes (Windows Vista/7/Server 2008)

  Be sure to log on as a user with administrator rights when 
  editing [Custom Paper Size] in the [Page Setup] sheet in the 
  printer driver.

================================================================

17. Access Permission to the Printer Properties 
    (Windows Vista/7/Server 2008)

  When you log on as a user with administrator rights who is 
  different from the one install the printer driver, you cannot 
  access the printer properties as a user with administrator 
  rights.
  To access the printer properties, perform either of the 
  following procedures.

  - Windows Vista and Server 2008
    Right-click the printer icon, and then select [Run as 
    administrator].

  - Windows 7 and Server 2008 R2
    Click [Change Properties] in the [General] sheet in the 
    printer properties.

================================================================

18. Printing Thin Lines (Windows Vista/7/Server 2008)

  In certain applications, thickness of a line can be set to a 
  value smaller than 1 pt but the actual thickness in the 
  printout may be equivalent to that of a 1 pt line.

================================================================

19. Cautions for When Upgrading from Windows Vista to Windows 7

  When upgrading the operating system from Windows Vista to 
  Windows 7, uninstall the printer driver first, upgrade the 
  operating system, and then reinstall the printer driver after 
  the upgrade.
  If you upgrade the operating system without uninstalling the 
  printer driver, the printer driver may not operate properly.
  In this case, uninstall the printer driver once, and then 
  reinstall the printer driver.
